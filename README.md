# 🎬 Cinematic Birthday Countdown Experience

A beautifully crafted **cinematic birthday website** designed to create an emotional and memorable experience. 
Built with **Vanilla HTML, CSS, and JavaScript**, this project heavily prioritizes animations, typography, and storytelling.

---

## ✨ Features

* 🎬 Cinematic intro screen
* ⏳ Live countdown timer
* ✍️ Emotional typography and layouts
* ❤️ Heartbeat animated popup
* ✨ Subtle visual effects (Custom dark "Bokeh" background)
* 📱 Fully responsive (mobile-first design)
* 🚧 Anticipation Rejection ("Not Yet") screen for early clicks

---

## 🛠️ Tech Stack

* HTML5
* Vanilla CSS (Variables, Flexbox/Grid, Keyframes)
* Vanilla JavaScript (ES6+)

---

## 📂 Project Structure

```
├── index.html       (Main Countdown View)
├── not-yet.html     (Early access interception screen)
├── style.css        (All styling & animations)
├── script.js        (Countdown timer & logic)
└── README.md
```

---

## 🚀 Getting Started

Since this is a lightweight, pure static frontend site, there is no need to run `npm install` or use a build step! Simply serve it locally:

```

### 1. Run a Local Server
You can simply use a lightweight HTTP server in the folder to view the site locally:
```
npx serve .
```
Navigate to **http://localhost:3000** in your browser.

*(Alternatively, you can just double-click `index.html` to open it directly!)*

---

## 🖼️ Customize

* Open `index.html` to edit customized messages or imagery.
* Open `script.js` to change the specific target countdown date.
* Update `style.css` to tweak primary gradient colors.
